# Tanmay Singla - Portfolio

My personal portfolio website showcasing professional experience and projects in software engineering, with a focus on cybersecurity and autonomous systems.

## Stack

- React + TypeScript
- Tailwind CSS
- Vite
- shadcn/ui

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
